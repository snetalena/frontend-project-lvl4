## Description
The objective of this project is to create a chat with the ability to add, remove and rename channels, using:

* React
* Redux (Redux Toolkit, Formik)
* Bootstrap React
* AJAX (Axios)
* Web Sockets
* Cookies
* Deploy (Heroku)
